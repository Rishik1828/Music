# 🎵 Emotion-Based Music Player

A fully offline, real-time system that detects your facial emotion via webcam and plays matching music from a local database — no external APIs, no cloud services.

---

## 📁 Project Structure

```
MUSICBEATS/
├── Dataset/
│   ├── train/          # FER2013 training images (7 emotion folders)
│   └── test/           # FER2013 test images
├── data/
│   └── songs.db        # SQLite song database (generated by setup_db.py)
├── models/
│   └── emotion_model.h5  # Trained CNN (generated by train.py)
├── src/
│   ├── preprocess.py   # FER2013 loader + augmentation + class weights
│   ├── train.py        # CNN training script
│   ├── detect.py       # Webcam + MTCNN face detection + inference + smoothing
│   ├── music_engine.py # SQLite query engine + pygame audio playback
│   ├── setup_db.py     # One-time DB setup (imports songs.csv.txt)
│   └── app.py          # Main Tkinter UI + integration
├── songs.csv.txt        # 420-song curated dataset
├── songs_schema.sql     # SQL schema + sample INSERTs
├── requirements.txt
└── README.md
```

## ⚙️ Installation

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Set up the song database
```bash
python src/setup_db.py
```
This reads `songs.csv.txt` and creates `data/songs.db` with all 420 songs.

### 3. Train the emotion model
```bash
python src/train.py
```
- Reads images from `Dataset/train/` and `Dataset/test/`
- Trains for up to 100 epochs with early stopping
- Saves the best checkpoint to `models/emotion_model.h5`
- Expected accuracy: **65–68% on FER2013 test set**

---

## 🎯 Dataset

**FER2013** — 35,887 grayscale 48×48 images, 7 emotion classes:

| Emotion  | Folder name |
|----------|-------------|
| Angry    | `angry/`    |
| Disgust  | `disgust/`  |
| Fear     | `fear/`     |
| Happy    | `happy/`    |
| Neutral  | `neutral/`  |
| Sad      | `sad/`      |
| Surprise | `surprise/` |

Place images under `Dataset/train/<emotion>/` and `Dataset/test/<emotion>/`.

## ▶️ Running the App

```bash
python src/app.py
```

1. Select your **Language** (Telugu / Hindi / English)
2. Select your **Era** (Old pre-2005 / New 2005-present)
3. Click **▶ Start** — the webcam activates
4. Your emotion is detected and music plays automatically
5. Press **Q** in the webcam window or **■ Stop** to quit

## 🎵 Adding Your Songs

The database stores `file_path` for each song. To wire up real audio:

```sql
UPDATE songs
SET file_path = 'C:/Music/Butta_Bomma.mp3'
WHERE title = 'Butta Bomma';
```

Or edit `songs.csv.txt` to add a `file_path` column before running `setup_db.py`.

## 🧠 Model Architecture

```
Input (48×48×1)
  Block 1: Conv64 → BN → ReLU → Conv64 → BN → ReLU → MaxPool → Dropout(0.25)
  Block 2: Conv128 → BN → ReLU → Conv128 → BN → ReLU → MaxPool → Dropout(0.25)
  Block 3: Conv256 → BN → ReLU → MaxPool → Dropout(0.25)
  Head:    Dense(512) → Dropout(0.5) → Dense(256) → Dropout(0.5) → Dense(7, softmax)
```

## 🔒 Constraints

- ✅ No external AI APIs
- ✅ No music streaming services
- ✅ Fully offline after setup
- ✅ All inference runs locally with `.h5` weights
- ✅ All songs are local `.mp3` / `.wav` files
